Airbnb High Booking Rates

Received Airbnb dataset with 95 columns of data which was cleaned and reduced to a usable format for various uses including text analysis and building statistical models.

Part I: Competing versus classmates on Kaggle to build the most accurate statistical model to predict the binary variable of whether a property would be high performing (high_booking) Model accuracy determined using Area Under the Curve (AUC) metric.

Part II: Researched the New Orleans market to build a model to provide insights into why certain properties are high booking within that specific market. Our team used 3 real Airbnb listings to pitch as a likely high booked property.

Programs: RStudio

Skills: ETL, Exploratory Data Analysis, Data Visualization, Machine Learning

Skills: Time Series Visualization, Financial Modelling, Stochastic Volatility Analysis, Predictive Analytics

Rainfall in Australia

In this project, we focused on classifying the binary variable RainTomorrow (whether it will rain the next day) using a variety of predictors. The machine learning models applied included logistic regression, K-Nearest Neighbors (KNN), decision tree-based methods (Random Forest, Bagging, XGBoost), and Support Vector Machines (SVM). Model performance was evaluated using key metrics such as confusion matrices, testing error rates, and Area Under the Curve (AUC).

Programs: RStudio

Skills: Machine Learning Models (Logistic Regression, K-Nearest Neighbors, Random Forest, Bagging, XGBoost, Support Vector Machines), Model Evaluation (Confusion Matrices, Testing Error Rates, AUC)
